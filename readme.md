# Natours

Proyecto de Node.js para la gestión de tours.

## Características

- API RESTful construida con Express
- Manejo de rutas y controladores
- Integración con MongoDB

## Instalación

```bash
git clone https://github.com/tu-usuario/natours.git
cd natours
npm install
```

## Uso

```bash
npm start
```
